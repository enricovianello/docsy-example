Released StoRM v1.11.21

Released packages: StoRM Backend 1.11.21, StoRM Frontend 1.8.15, StoRM WebDAV 1.4.1 and StoRM Utils 1.0.0
Wednesday, May 12, 2021

The StoRM Product Team is pleased to announce the release of StoRM 1.11.21 that includes the following updated components:

  • StoRM Backend v. 1.11.21
  • StoRM Frontend v. 1.8.15
  • StoRM WebDAV v. 1.4.1

and introduces:

  • StoRM Utils v. 1.0.0

This release:

  • fixes the known issue about the upgrade to StoRM v1.11.20 which could break connections with MariaDB
  • fixes the boot order for both Frontend and Backend ensuring that mariadb service is started before StoRM services;
  • fixes the failed state shown on stop/restart of Java components;
  • provides a set of scripts that can be used to edit from command line the storage space info related to a storage area.

Please, follow the upgrade instructions. Read the release notes for more details.
